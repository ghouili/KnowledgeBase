Glossaire des termes clésGlossary of key terms

Les définitions essentielles du module, du matériel au logiciel. Tapez pour filtrer.The module's essential definitions, from hardware to software. Type to filter.

BIOS / UEFI
Firmware d'amorçage stocké en ROM qui initialise le matériel au démarrage. L'UEFI (moderne) remplace le BIOS : interface graphique, disques > 2 To (GPT), Secure Boot, Fast Boot.Boot firmware stored in ROM that initialises hardware at startup. UEFI (modern) replaces BIOS: graphical UI, disks > 2 TB (GPT), Secure Boot, Fast Boot.
Bootloader
Programme qui charge le noyau du SE en mémoire (GRUB sous Linux, Windows Boot Manager). Affiche un menu si plusieurs SE sont installés.Program that loads the OS kernel into memory (GRUB on Linux, Windows Boot Manager). Shows a menu when several OSes are installed.
CPU Central Processing Unit
Le « cerveau » qui exécute les instructions via le cycle Fetch-Decode-Execute. Contient UC, UAL, registres et caches L1/L2/L3.The "brain" that runs instructions through the Fetch-Decode-Execute cycle. Contains the control unit, ALU, registers and L1/L2/L3 caches.
CLI Command Line Interface
Interface en ligne de commande : interaction par texte (cmd, bash, PowerShell). Rapide, scriptable, idéale pour les serveurs.Command-line interface: text-based interaction (cmd, bash, PowerShell). Fast, scriptable, ideal for servers.
DDR Double Data Rate
Type de RAM moderne qui transfère deux fois par cycle d'horloge. Générations DDR à DDR5 (67 Go/s), non compatibles entre elles.Modern RAM type transferring twice per clock cycle. Generations DDR through DDR5 (67 GB/s), not cross-compatible.
DRAM / SDRAM
DRAM : RAM dynamique dont chaque bit est rafraîchi périodiquement. SDRAM : DRAM synchronisée avec l'horloge du bus système.DRAM: dynamic RAM whose every bit is refreshed periodically. SDRAM: DRAM synchronised with the system-bus clock.
Pilote (driver)Driver
Petit programme qui fait le pont entre le SE générique et un matériel précis. Sans le bon pilote, le périphérique ne fonctionne pas.Small program bridging the generic OS and a specific device. Without the right driver, the peripheral won't work.
EULA End User License Agreement
Contrat d'utilisation des logiciels propriétaires, accepté à l'installation.End-user agreement for proprietary software, accepted at install time.
GUI Graphical User Interface
Interface graphique : fenêtres, icônes, menus, souris (Explorateur, GNOME, Finder). Intuitive et visuelle.Graphical interface: windows, icons, menus, mouse (Explorer, GNOME, Finder). Intuitive and visual.
Hardware
Le matériel — tout ce qu'on peut toucher : carte mère, CPU, RAM, disques, GPU, périphériques.The physical machine — everything you can touch: motherboard, CPU, RAM, drives, GPU, peripherals.
HDD / SSD / NVMe
HDD : disque mécanique à plateaux. SSD : mémoire flash sans pièce mobile. NVMe : SSD sur bus PCIe (3–14 Go/s).HDD: mechanical platter disk. SSD: flash memory with no moving parts. NVMe: SSD on the PCIe bus (3–14 GB/s).
Carte mèreMotherboard
Circuit principal qui interconnecte tous les composants et coordonne la communication via le chipset. Formats : ATX, Micro-ATX, Mini-ITX.Main board interconnecting all components and coordinating communication via the chipset. Form factors: ATX, Micro-ATX, Mini-ITX.
POST Power-On Self-Test
Auto-test au démarrage : le firmware vérifie RAM, CPU, GPU et clavier. Une erreur génère des bips ou un code.Self-test at boot: the firmware checks RAM, CPU, GPU and keyboard. An error triggers beeps or a code.
RAM Random Access Memory
Mémoire vive de travail : volatile, rapide, lecture/écriture. Contient le noyau, les programmes ouverts et les données. Disparaît à l'extinction.Working memory: volatile, fast, read/write. Holds the kernel, open programs and live data. Lost at power-off.
ROM Read Only Memory
Mémoire morte non volatile, en lecture seule. Variantes : PROM, EPROM, EEPROM, Flash. Contient le BIOS/UEFI et les firmwares.Non-volatile, read-only memory. Variants: PROM, EPROM, EEPROM, Flash. Holds the BIOS/UEFI and firmware.
SemVer Semantic Versioning
Convention de version MAJOR.MINOR.PATCH (ex. 3.11.4). MAJOR = casse, MINOR = ajout compatible, PATCH = correctif.Versioning convention MAJOR.MINOR.PATCH (e.g. 3.11.4). MAJOR = breaking, MINOR = compatible add, PATCH = fix.
SHA-256
Fonction de hachage utilisée pour vérifier l'intégrité d'un fichier téléchargé (détecte toute altération).Hash function used to verify the integrity of a downloaded file (detects any tampering).
Software
Le logiciel — code et données : système d'exploitation, pilotes et applications.The coded layer — code and data: operating system, drivers and applications.
Système d'exploitation (SE)Operating System (OS)
Couche logicielle entre le matériel et les applications : gère mémoire, fichiers, processus et périphériques (Windows, Linux, macOS).Software layer between hardware and applications: manages memory, files, processes and devices (Windows, Linux, macOS).
VLSI
Very Large Scale Integration — intégration de millions de transistors sur une puce, base des microprocesseurs (4ᵉ génération).Very Large Scale Integration — millions of transistors on one chip, the basis of microprocessors (4th generation).
winget / apt
Gestionnaires de paquets en ligne de commande : winget (Windows), apt (Debian/Ubuntu). Installent, mettent à jour et désinstallent des logiciels.Command-line package managers: winget (Windows), apt (Debian/Ubuntu). Install, update and remove software.
