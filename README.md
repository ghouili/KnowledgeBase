# Systèmes d'exploitation — Installer des composants matériels et logiciels

Interactive, bilingual (FR/EN) course companion for the **Operating Systems** module by **Walid Ghouili** — CTO & Founder at InnoviaBurst, technical mentor & lecturer at **IMSET**.

🔗 **Live site:** https://ghouili.github.io/KnowledgeBase/

## Features

- 🌍 Bilingual French / English with a one-click toggle (preference saved)
- 🌗 Light / dark mode
- 📱 Fully responsive, accessible academic design
- 📥 Downloadable slide decks per chapter
- ✅ Interactive quizzes and practical exercises

## Publish on GitHub Pages

1. Push these files to the repository root (branch `main`).
2. Repo **Settings → Pages → Build and deployment → Source: Deploy from a branch**.
3. Branch: `main`, folder: `/ (root)` → **Save**.
4. The site goes live at `https://ghouili.github.io/KnowledgeBase/`.

> `.nojekyll` is included so GitHub Pages serves all files as-is.
