Introduction

Brancher un câble USB ne suffit pas toujours. De nombreux périphériques requièrent un pilote spécifique, une vérification de compatibilité, la lecture de la documentation, une configuration et un test final. Ce chapitre donne une méthodologie pour installer n'importe quel périphérique avec confiance.Plugging in a USB cable is not always enough. Many peripherals require a specific driver, a compatibility check, reading the documentation, a configuration step and a final test. This chapter gives a methodology to install any peripheral with confidence.

2 · Le pilote (driver)2 · The driver

Un pilote est un petit programme qui sait « parler » à un matériel précis. Il fait le pont entre le SE générique (« j'ai un fichier à imprimer ») et le matériel précis (qui attend un format de commande spécifique). Sans le bon pilote, le SE ne peut pas utiliser le périphérique.A driver is a small program that knows how to "speak" to a specific piece of hardware. It bridges the generic OS ("I have a file to print") and the precise device (which expects a specific command format). Without the right driver, the OS cannot use the peripheral.

Chaîne : Application → API du SE → Pilote → Périphérique physique. Origines : intégré au SE (~80 %), site du constructeur, CD/clé, Windows Update, modules kernel (Linux).Chain: Application → OS API → Driver → Physical device. Sources: built into the OS (~80%), maker's website, CD/USB, Windows Update, kernel modules (Linux).

5 · Check-list de compatibilité5 · Compatibility checklist

  1. Système — le pilote existe-t-il pour votre OS ?OS — does the driver exist for your OS?
  2. Architecture CPU — x86_64 ou ARM ?CPU architecture — x86_64 or ARM?
  3. Version du SE — Windows 10 22H2 ? Ubuntu 22.04 LTS ?OS version — Windows 10 22H2? Ubuntu 22.04 LTS?
  4. Port disponible — USB-A/C, RJ45, Bluetooth, HDMI ?Available port — USB-A/C, RJ45, Bluetooth, HDMI?
  5. Espace disque — souvent 500 Mo à 2 Go.Disk space — often 500 MB to 2 GB.
  6. RAM — 8–16 Go pour certains logiciels pro.RAM — 8–16 GB for some pro software.
  7. Alimentation — externe requise ? tension ?Power — external supply needed? voltage?
  8. Réseau — Wi-Fi 2.4/5 GHz, IPv4/IPv6, Internet requis ?Network — Wi-Fi 2.4/5 GHz, IPv4/IPv6, Internet needed?
