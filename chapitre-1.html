Introduction au moduleIntroduction to the module

Tout technicien informatique — développeur, administrateur, support ou usager averti — doit savoir identifier les composants d'un ordinateur, installer et configurer un système d'exploitation, raccorder des périphériques, installer des logiciels en respectant la documentation, et diagnostiquer les pannes courantes. Every IT technician — developer, administrator, support engineer or power user — must be able to identify a computer's components, install and configure an operating system, connect peripherals, install software in line with its documentation, and diagnose common faults.

You will learn to read technical documentation, verify minimum requirements and plan installation steps before acting — across 4 labs spread over the 4 chapters.

3 · Hardware vs Software3 · Hardware vs Software

Un ordinateur repose sur deux piliers indissociables.A computer rests on two inseparable pillars.

🔩 Hardware — le matérielthe physical

Tout ce qu'on peut toucher : carte mère, CPU, RAM, ROM, disques (SSD/HDD), GPU, cartes réseau/son, alimentation, boîtier, périphériques. Sans hardware → pas d'ordinateur.Everything you can touch: motherboard, CPU, RAM, ROM, drives (SSD/HDD), GPU, network/sound cards, PSU, case, peripherals. No hardware → no computer.

💾 Software — le logicielthe coded

Tout ce qu'on ne peut pas toucher — code et données : système d'exploitation (Windows, Linux, macOS), pilotes (drivers), applications. Sans software → un tas de circuits inertes.Everything you cannot touch — code and data: operating system (Windows, Linux, macOS), drivers, applications. No software → a pile of inert circuits.

La carte mère — colonne vertébraleThe motherboard — the backbone

La carte mère interconnecte tous les composants, distribue le courant et coordonne la communication via le chipset. Elle ne calcule pas — elle relie.The motherboard interconnects every component, distributes power and coordinates communication through the chipset. It does not compute — it links.

Le processeur (CPU) — le cerveauThe processor (CPU) — the brain

Un CPU contient l'UC (unité de contrôle), l'UAL/ALU (calculs), des registres et des caches L1/L2/L3. Il répète le cycle Fetch → Decode → Execute des milliards de fois par seconde.A CPU contains the control unit, the ALU (arithmetic/logic), registers and L1/L2/L3 caches. It repeats the Fetch → Decode → Execute cycle billions of times per second.

Caractéristiques clés : fréquence (GHz), nombre de cœurs, taille du cache. Constructeurs : Intel, AMD (PC) ; ARM, Apple (mobile, Mac).Key specs: clock speed (GHz), core count, cache size. Makers: Intel, AMD (PC); ARM, Apple (mobile, Mac).

5 · Les mémoires : morte & vive5 · Memory: ROM & RAM

ROM — Read Only Memory

Mémoire morte / firmware. Non volatile, en lecture seule, très petite. Types : ROM, PROM, EPROM, EEPROM, Flash. Usage : BIOS/UEFI, firmware des périphériques.Read-only memory / firmware. Non-volatile, read-only, very small. Types: ROM, PROM, EPROM, EEPROM, Flash. Use: BIOS/UEFI, peripheral firmware.

RAM — Random Access Memory

Mémoire vive / mémoire de travail. Volatile (disparaît à l'extinction !), lecture/écriture, grande (4–128 Go), rapide. Contient le noyau, les programmes ouverts et les données.Working memory. Volatile (lost at power-off!), read/write, large (4–128 GB), fast. Holds the kernel, open programs and live data.

RAM ≠ Stockage. La RAM disparaît à l'extinction ; le disque conserve. C'est pourquoi on sauvegarde sur disque.RAM ≠ Storage. RAM is lost at shutdown; the disk persists. That is why we save to disk.

6 · Classification des périphériques6 · Classifying peripherals

Un périphérique est tout matériel raccordé à l'ordinateur. Trois classifications principales :A peripheral is any hardware connected to the computer. Three main classifications:

Par directionBy direction

Entrée : clavier, souris, scanner, webcam. Sortie : écran, imprimante, haut-parleurs. E/S : écran tactile, disque, clé USB.Input: keyboard, mouse, scanner, webcam. Output: screen, printer, speakers. I/O: touchscreen, disk, USB stick.

Par localisationBy location

Internes : RAM, SSD, GPU. Externes : clavier USB, imprimante. Embarqués : écran/webcam de laptop.Internal: RAM, SSD, GPU. External: USB keyboard, printer. Built-in: laptop screen/webcam.

Par interfaceBy interface

Filaire : USB, HDMI, Ethernet, SATA. Sans fil : Wi-Fi, Bluetooth.Wired: USB, HDMI, Ethernet, SATA. Wireless: Wi-Fi, Bluetooth.
