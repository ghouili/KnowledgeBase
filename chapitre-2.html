IntroductionIntroduction

Une fois le matériel installé, il faut savoir l'utiliser. Ce chapitre couvre tout le cycle d'utilisation : comprendre le démarrage, manipuler le système via ses deux interfaces (ligne de commande et graphique), gérer fichiers et dossiers, configurer les paramètres fondamentaux et personnaliser l'environnement.Once the hardware is installed, you must know how to use it. This chapter covers the full usage cycle: understanding boot-up, operating the system through its two interfaces (command line and graphical), managing files and folders, configuring fundamental settings and customising the environment.

5 · CLI vs GUI5 · CLI vs GUI

CLI — ligne de commandecommand line

+ rapide, scriptable, léger, accès à distance (SSH), toutes les options. − courbe d'apprentissage, erreurs destructrices, pas de visualisation. Ex. : cmd, bash, PowerShell.+ fast, scriptable, lightweight, remote (SSH), every option. − learning curve, destructive typos, no visualisation. e.g. cmd, bash, PowerShell.

GUI — interface graphiquegraphical interface

+ intuitif, visualisation directe, drag & drop, multi-fenêtres. − plus lent pour répéter, consomme RAM/CPU, pas toujours scriptable. Ex. : Explorateur, GNOME, Finder.+ intuitive, direct visualisation, drag & drop, multi-window. − slower to repeat, uses RAM/CPU, not always scriptable. e.g. Explorer, GNOME, Finder.

Le mieux : maîtriser les deux. GUI pour le quotidien, CLI pour l'efficacité et les serveurs.Best of all: master both. GUI for everyday work, CLI for efficiency and servers.

7 · Configurer & personnaliser un SE7 · Configure & customise an OS

Après installation, sept réglages prioritaires :After installation, seven priority settings:

  1. Région & langue — langue, format date/heure, clavier.Region & language — language, date/time format, keyboard.
  2. Fuseau & heure — synchroniser via NTP (logs, SSL, certificats).Time zone & clock — sync via NTP (logs, SSL, certificates).
  3. Réseau — Wi-Fi/Ethernet, IP statique ou DHCP, DNS (8.8.8.8, 1.1.1.1).Network — Wi-Fi/Ethernet, static IP or DHCP, DNS (8.8.8.8, 1.1.1.1).
  4. Compte — mot de passe fort, 2FA ; éviter l'usage quotidien du compte admin.Account — strong password, 2FA; avoid daily use of the admin account.
  5. Mises à jour — Windows Update ou apt update && apt upgrade.Updates — Windows Update or apt update && apt upgrade.
  6. Antivirus / pare-feu — Defender (Windows), ufw (Linux).Antivirus / firewall — Defender (Windows), ufw (Linux).
  7. Sauvegardes — automatiques ; tester la restauration avant d'en avoir besoin.Backups — automatic; test restoration before you ever need it.
Personnalisation Windows : thème clair/sombre, fond d'écran, applis au démarrage (shell:startup), programmes par défaut, accessibilité, options d'alimentation, variables d'environnement, regedit (⚠ exporter avant de modifier).Windows customisation: light/dark theme, wallpaper, startup apps (shell:startup), default programs, accessibility, power options, environment variables, regedit (⚠ export before editing).
